|travis-img| pageone ====== *a module for polling urls and stats from
homepages*

Install
-------

::

    mkvirtualenv pageone
    git clone https://github.com/newslnyx/pageone.git
    cd pageone
    pip install -r requirements.txt
    pip install .

Test
----

Requires ``nose``

::

    nosetests
